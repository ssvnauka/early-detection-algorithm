# Алгоритм раннего выявления рака желудка

## Диаграмма алгоритма
```mermaid
graph TD
    A[Оценка факторов риска] --> B{Высокий риск?}
    B -- Нет --> C[Мониторинг раз в 3 года]
    B -- Да --> D[Гастроскопия]

    G --> H{Рак подтвержден?}
    H -- Да --> I[Терапия и наблюдение]
    H -- Нет --> J[Повторное обследование через 6 мес]
